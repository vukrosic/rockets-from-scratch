# Lesson 03: Thrust And Impulse

Thrust is a force.

Impulse is force accumulated over time.

This lesson connects those two ideas.

## Step 1: Thrust As A Force

Thrust is measured in newtons:

```text
N = kg m/s^2
```

Thrust is what the engine contributes to the rocket's net force.

If thrust is greater than the downward forces, the rocket can accelerate
upward.

## Step 2: Impulse

Impulse is:

```text
J = F Delta t
```

Where:

```text
J:
    impulse

F:
    force

Delta t:
    time interval
```

Impulse units:

```text
N s
```

Impulse tells you how much momentum changes:

```text
J = Delta p
```

That means a force applied for longer time produces a bigger momentum change.

## Step 3: Average Thrust

If thrust changes during a burn, the average thrust is:

```text
average thrust = total impulse / burn time
```

Example:

```text
total impulse = 200 N s
burn time = 4 s
```

Then:

```text
average thrust = 200 / 4 = 50 N
```

Peak thrust and average thrust are not the same thing.

## Step 4: Thrust Curve

A thrust curve is force as a function of time.

It might look like:

```text
rise
peak
fall
```

The area under the thrust curve is total impulse.

That is the same area idea you saw in motion graphs.

## Step 5: Total Impulse

If thrust is constant:

```text
J = F t
```

Example:

```text
F = 80 N
t = 5 s
```

Impulse:

```text
J = 80 * 5 = 400 N s
```

Meaning:

```text
the engine changes momentum by 400 kg m/s
```

## Step 6: Why Impulse Matters

Two engines can have the same average thrust but different burn times.

One can deliver a quick shove.

Another can deliver a long push.

Both can have the same total impulse.

That matters because mission goals are different.

Some rockets need a strong launch push.

Some need long, efficient burns in space.

## Step 7: Tiny Hand Check

A motor produces:

```text
average thrust = 30 N
burn time = 6 s
```

Total impulse:

```text
J = F t
J = 30 * 6
J = 180 N s
```
